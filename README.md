
## 📌 Project Stappenplan

### 1️⃣ Doelen en Scope
#### 🎯 Doel van de Software
Een gebruiksvriendelijke website waar gebruikers AI-modellen kunnen downloaden voor entertainment doeleinden. De AI-modellen worden met een interface in the terminal gedraaid.  

#### 🔥 Belangrijkste Functionaliteiten
✅ Gebruiksvriendelijke UI en navigatie  
✅ Duidelijke communicatie tussen speler en AI  
✅ Tekst en spraak interactie met AI  
✅ Websockets voor snelle API-verbindingen  
✅ Duidelijke integratie en uitleg van de AI API

#### 📌 MVP Scope
✔️ Basis UI en navigatie  
✔️ TTS en STT ondersteuning  
✔️ Connectie met OpenAI GPT-4o mini   
✔️ Chatgeschiedenis opslaan binnen sessie  
✔️ Prompt bewerking door gebruiker   
✔️ Websockets voor API-verbinding   
✔️ Download optie voor de applicatie   
❌ Geen chatbot op de site zelf  

---

### 2️⃣ User Stories + Functionaliteiten
| **Functionaliteit**           | **User Story** | **Acceptatiecriteria** |
|------------------------------|---------------|-----------------------|
| **Basis UI & Navigatie** | "Als klant wil ik een basis UI en navigatie, zodat ik gemakkelijk kan navigeren." | Consistent ontwerp, navigatiebalk op elke pagina, werkende routing. |
| **Contact opnemen** | "Als klant wil ik contact kunnen opnemen, zodat ik vragen kan stellen of problemen kan melden." | Werkend contactformulier met validatie. |
| **TTS & STT** | "Als klant wil ik AI via tekst en spraak kunnen bedienen." | Werkende integratie met API's, foutmeldingen en meldingen bij activatie. |
| **GPT-4o mini Connectie** | "Als klant wil ik met AI kunnen chatten." | Berichten verzenden en ontvangen via API, makkelijke API-key invoer. |
| **Chatgeschiedenis** | "Als klant wil ik mijn gesprekken kunnen teruglezen." | Minimaal laatste X berichten opslaan, chronologische weergave. |
| **Prompt Bewerking** | "Als klant wil ik de AI prompt kunnen aanpassen." | Bewerken via apart bestand, documentatie erbij, standaard terugkeeroptie. |
| **Websockets voor API's** | "Als ontwikkelaar wil ik real-time communicatie met de AI." | Stabiele websocketverbinding, automatische reconnect, foutafhandeling. |
| **Download optie** | "Als klant wil ik de applicatie kunnen downloaden." | Downloadknop aanwezig, juiste bestandstype (.exe). |
